Help for typewriter

What is typewriter?

Typewriter is a simple program written in Java programming language. I wrote it because I wasn't happy with other novel and poem writing software. Then I decided to publish since other people might find it useful for their writing.

Typewriter is a free software. This means you can modify it and/or distributed as you want. However, you have to keep the license, which GNU General Public License 3.0, you have to state your changes, keep the program open-source and keep link to the original author, who is uhl1k (Roman Janků).

Source codes and other information about Typewriter can be found on its public GitHub repository.

Description of interface

There are four main part of the Typewriter graphical interface. First is menu bar at the very top. Here you can find options for interacting with files under the File menu and information about the program and its license under the Help menu.

Second important part of the interface is toolbar. Its default position is immediately under the menu, however you can move it around as you wish using the mouse. On this toolbar there are buttons for creating, modifying and deleting the books, chapters and poems as well as export buttons.

Third part if the interface are two boxes on the left. The top one is list of the books. All books created in the opened file will be shown here. The bottom box shows chapters and poems that are in the selected book. You can change size of these boxes by pulling their edges.

The last part of the interface is the text area in the rest of the window. In this area the content of selected chapter or poem is shown and you can modify it.

Create a new file

When you start the Typewriter you already have a new file created. However, if you have another file already opened, you need to create new file manually. Go to File menu and click on Open option. If you have any unsaved changes, the program will ask you if you want to save them. Then a list of books and chapters and poems and the editing area will be cleaned. At this point you have a new file opened. But be careful, this file is only in RAM memory. You need to save it to keep the data after you close the Typewriter.

Open an existing file

You have two options when it comes to opening files. Either you can supply file name as an argument when starting the Typewriter. The file will be automatically opened when the Typewriter starts.

The second option is to go into File menu and click Open option. A dialog will appear in which you can select the file you want to open. After confirming the dialog the selected file will be opened. If you have any unsaved changes you will be asked if you want to save them.

Creating a book

To create a book click on the Add new book button in the toolbar.

Typewriter will ask you for the name of the new book. Enter the name and confirm the dialog. The new book will be created and will be added to the list of books. You can now work with it and add chapters and poems to it or export it.

Go back up

Adding poems and chapters

To add a new chapter to a book, click on the Add new chapter button in the toolbar to add new chapter

or alternatively use Add new poem button in the toolbar to add new poem to a book.

You need to select a book before adding chapter or poem. Typewriter will ask you for a name of the chapter or poem. After confirming the name the chapter or poem will be created and added to the book. You will see it in the list of poems and chapters after selecting the book.

Go back up

Modifying book, chapters and poems

To modify book click on the Modify book button in the toolbar.

You can change the title and author of the book. To save changes click on Save button. If you do not want to save changes close the dialog using x in the top right corner.

To modify chapter or poem click on the Modify chapter/poem button in the toolbar.

In the dialog that will open you can modify name of the chapter or poem. Content of the chapter is displayed in the text area in the main window and you can edit it directly.

To change the order of chapters and poems in the book you can use the Move up and Move down buttons in the toolbar.

Select the chapter or poem you want to move and move it to the place you want it to be using the buttons.

Go back up

Deleting books, chapters and poems

To delete a book select the book you want to delete and click on Delete book button in the toolbar.

You will be asked for confirmation and the book will be deleted including all its poems anc chapters. To delete a poem or chapter select the poem or chapter you want to delete and click on the Delete chapter/poem button in the toolbar.

Selected chapter or poem will be deleted.

Go back up

Saving file

To save changes in the file use option Save in the File menu. If the file has not been saved yet you will be asked for the file name and location you want to save it in.

To save changes in a new file us option Save as in the File menu. You will be asked for name and location of the file.

Go back up

Exporting book

To export the book and all its chapters and poems to a file use Export buttons at the end of a toolbar.

The file extension written on the button marks the format in which the export will be. So .txt means plain text file, .tex means latex file and .rtf mean Rich-Text format.
